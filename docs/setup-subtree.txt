Create a folder for the separate repo. Add a few files if desired.

Create an empty repo on github or equivilant.

Add a remote:
git remote add <remote-name> <repository-url>
(e.g. git remote add bootstrap git@github.com:dommcdev/bootstrap.git)

Push files from private repo to public repo for the first time (will have to be repeated for each change)
git subtree push --prefix=<folder-name> <remote-name> <remote-branch>
(e.g. git subtree push --prefix=public bootstrap main)


(To pull changes from public repo, do)
git subtree pull --prefix=public bootstrap main --squash
